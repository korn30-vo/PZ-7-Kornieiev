# Laboratory Work №7
Study of Unsupervised Learning Methods
Practical work completed by student of group PDM-51
Kornieiev Volodymyr


In modern data analysis and machine learning, regression models are widely used for solving prediction tasks involving numerical values. They allow identifying relationships between variables and building mathematical models to predict outcomes based on input data.
The purpose of this laboratory work is to study and practically implement linear, multiple, and polynomial regression methods, as well as to evaluate model performance using MSE and R² metrics. Additionally, methods for improving models are considered, such as cross-validation, overfitting prevention, and hyperparameter tuning using GridSearch.
